== README

This is a sample app to demonstrate how to add Gravatars to Rails apps. It is a simple scaffolded app with a user model. All steps are in the "code added for gravatar" commit and marked with comments.

It was prepared for a Lightning Talk at Seattle.rb. The slides can be found at:

http://www.slideshare.net/ElizabethKari/adding-gravatars-to-rails-apps
